# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.12.x  | ✅        |
| < 0.12  | ❌        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them privately via [GitHub's private vulnerability reporting](https://github.com/w8s/w8s-astro-mcp/security/advisories/new).

I'll acknowledge within a few days and aim to release a fix within 30 days depending on severity.
